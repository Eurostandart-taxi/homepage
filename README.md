# Eurostandart Taxi Homepage
This is the homepage for EuroStandart Taxi, a service partner with Yandex Go. Visit https://eurostandarttaxi.com to see the details.


# About this code
The code in this repo is used by Cloudflare Workers. Any updates to the `main` branch will automatically be deployed to the workers and become visible at https://eurostandarttaxi.com/.

The `assets` directory contains all necessary assets. Breba code automatically uploads necessary assets to a CDN for better performance. Therefore, changing files in the `assets` directory is for the record only, and changing them in GitHub will not change them on the website.
